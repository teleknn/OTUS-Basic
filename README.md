# Коментарии по проекту 
```bash
Как все работает?

Я беру цифру и перевожу ее в массив, каждая цифра в отдельную ячейку массива. (arrayNumbers).

Далее беру по 3 цифры (разряд – digit),  (для этого есть специальный метод который берет нужный мне разряд arrayTakeDigitPart)
и “прогоняю” его через специальный набор методов который в рамках одного разряда заменяет цифры на буквы. Так же в случае 
с  тысячами и подобными добавляет – тысяч, миллионов и так далее.
Учитывается  род разряда так как тысяча – ОНА, миллион – ОН.
Я сделал методы void т.е. они сразу выводят полученный результат на экран.
Но в целом можно легко переделать чтобы результат так же записывался в массив или в коллекцию, 
например для дальнейшего использования.
```
